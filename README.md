# cpp-cmd-calculator

A basic cmd calculator, my first project made in C++

# Operators

* \+ Addition
* \- Subtraction
* \* Multiplication
* \/ Division
* \^ Exponent
* \% Modulus

# Functions

* sqrt(...) - Square Root
* cbrt(...) - Cube Root
